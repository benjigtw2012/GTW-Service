# Window Survey App

Vercel-ready Next.js mobile survey app.

## Deploy
1. Upload the contents of this folder to GitHub.
2. Import the repo in Vercel.
3. Framework: Next.js.
4. Build command: `npm run build`.
5. Output directory: leave blank.

## If you see `Unexpected end of JSON input`
- Make sure GitHub contains the app files, not just a ZIP file.
- Delete any empty `package-lock.json` or broken lockfile if one was accidentally created.
- Re-deploy from Vercel.
- If it happens in the browser, clear the site's storage/localStorage and reload.
